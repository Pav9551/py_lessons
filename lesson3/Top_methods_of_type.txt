1. списки (list):
len(my_list) - длина списка
my_list.clear() - очищает список
my_list.append() - добавляет элемент в конец списка
my_list.sort() - сортирует список
my_list.pop() - удаляет элемент с индексом i и возвращает

2. словари (dict):
len(my_list) - длина словаря
my_list.clear() - очищает словарь
my_dict.update() - обновляет словарь данными
my_dict.items() - возвращает пары (ключ, значение)
my_dict.pop(key[, default]) - удаляет ключ и возвращает значение.

3. множества (set)
len(my_set) - количество элементов в множестве
my_set.clear() - очищает множество
my_set.add() - добавляет элемент в множество
my_set.intersection()- пересечение множеств
my_set.discard() - удаляет элемент из множества, если он там есть

4. строки (str)
len(my_str) - количество символов в строке
my_str1 + my_str2 - конкатенация (сложение строк)
my_str.find(str, [start],[end]) - поиск в строке
my_str.lower() - переводит в нижний регистр
my_str.format(*args, **kwargs) - форматирование строки
